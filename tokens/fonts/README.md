# Gebündelte Schriften

Diese Schriftdateien werden vom `swiss`-Theme genutzt (Design System der
Schweizer Bundesverwaltung, https://swiss.github.io/designsystem) und über
`tokens/fonts.css` (@font-face) eingebunden.

## Noto Sans

- `NotoSans-Regular.ttf`, `NotoSans-Bold.ttf`, `NotoSans-Italic.ttf`,
  `NotoSans-BoldItalic.ttf`
- Lizenz: **SIL Open Font License 1.1** (OFL) — freie Weitergabe und Bündelung
  erlaubt.
- Herkunft: Google Noto Sans (dieselbe Familie, die das Swiss Design System als
  Hausschrift ausliefert).

Die Bündelung hält das `swiss`-Theme offline- und markentreu, ohne externe
Font-CDNs (keine Netzabhängigkeit zur Laufzeit).
